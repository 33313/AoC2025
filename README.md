## Advent of Code 2025
This repo contains all of my AoC 2025 solutions.

To run the solutions:

0. [Install Go](https://go.dev/doc/install)
1. Clone this repository and `cd` into it
2. `cd` into any `day_xx` directory
3. Run `go run main.go`

**Obviously, there are spoilers ahead.** Try to solve the puzzles on your own first!
